# 平台变量文档(数据加工平台)

平台在作业执行期会把 SQL 里的 `${...}` 占位变量替换成实际值。生成 SQL 时原样保留占位符,不要替换成具体日期。

## 可用变量

| 变量 | 执行期替换为 | 用法 |
|---|---|---|
| `${TX_DATE}` | 作业实例执行日的前一天(`YYYY-MM-DD`) | T+1 增量过滤:`WHERE tx_date = '${TX_DATE}'` |
| `${TX_DATE_NODASH}` | `${TX_DATE}` 的无横杠形式(`YYYYMMDD`) | 分区/命名拼接 |
| `${RUN_DATE}` | 执行当天日期(`YYYY-MM-DD`) | 记录加工时间 |

## 约定

- 日期变量带单引号当字符串用于比较:`WHERE stat_date = '${TX_DATE}'`。
- T+1 增量作业只处理 `${TX_DATE}` 当天的数据,不要全表全量扫描。
